- Core documentation (business plans, SOPs, brand guidelines)
- Products/services (code, content, operations)
- Content pipeline (drafts, published, assets)
- Automation (workflows, scripts)

## Folder Structure

Each business follows the same pattern:
```
BusinessName/
├── 00_Core/           Business docs, SOPs, brand
├── 01_Product/        Main product/service
├── 02_Operations/     Finance, payroll, admin
├── 03_Content/        Marketing & social media
└── 04_Automation/     Scripts & workflows
```

This consistency lets you scale businesses without rebuilding infrastructure.
