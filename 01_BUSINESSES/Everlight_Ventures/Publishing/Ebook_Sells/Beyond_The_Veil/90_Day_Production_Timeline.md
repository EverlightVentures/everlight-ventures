# Beyond the Veil -- 90-Day Empire Launch Plan

**Start Date:** TBD (fill in when manuscript is finalized)
**Goal:** Audio-first launch with ebook + print simultaneous, followed by social clip drip

---

## Phase 1: Foundation (Days 1-30)
*Audio production + cover art + metadata prep*

### Week 1-2: Pre-Production Lock
- [ ] Finalize manuscript -- no edits after this point
- [ ] Write back-cover blurb (250-400 words; see metadata_sync.py TODO)
- [ ] Select 7 Amazon keywords (use Publisher Rocket or KDP autocomplete)
- [ ] Choose KDP categories (primary + secondary BISAC codes)
- [ ] Commission cover art (hire on 99designs or Reedsy marketplace; ~$200-600)
- [ ] Post ACX audition listing with test passage from Voice_Brand_SOP.md
- [ ] Apply for ISBNs via Bowker (bowker.com; ~$125 for 10 ISBNs)

### Week 3-4: Narrator + Production
- [ ] Receive audition demos -> score with scorecard in Narrator_Selection README
- [ ] Select narrator; negotiate contract (royalty share vs PFH -- see SOP)
- [ ] Narrator records Chapter 1 + Chapter 2 for approval before full production
- [ ] Review against quality checklist in Voice_Brand_SOP.md
- [ ] Begin full audiobook recording (narrator self-directs with SOP in hand)
- [ ] Run metadata_sync.py to generate platform files (fill TODOs first)
- [ ] Set up KDP account + ACX account if not already done

---

## Phase 2: Production + Pre-Launch (Days 31-60)
*All assets complete; build hype machine*

### Week 5-6: Asset Finalization
- [ ] Audiobook recording complete (all chapters)
- [ ] Audio QC: check every chapter against ACX technical specs
- [ ] Mastering pass: normalize, de-noise, final mix
- [ ] Run clip_slicer.sh -- generate 10-15 social clips from best moments
- [ ] Add subtitles to clips (CapCut, Descript, or similar)
- [ ] Format ebook (Vellum or Reedsy ebook formatter; ~$0-$200)
- [ ] Format print interior PDF (KDP print templates; free)
- [ ] Cover art final delivery: separate files for ebook, print (with spine), audio

### Week 7-8: Platform Setup + Pre-Orders
- [ ] Upload ebook to KDP -- set 90-day Kindle Select (Unlimited) enrollment
- [ ] Upload print to KDP (paperback first)
- [ ] Upload to IngramSpark (hardcover + wide distribution paperback)
- [ ] Upload audio to ACX -- submit for Audible review (takes 1-3 weeks to approve)
- [ ] Set release date: 2-3 weeks out to allow pre-order window
- [ ] Create author page on Amazon Author Central
- [ ] Set up email capture landing page (Mailchimp or ConvertKit free tier)
- [ ] Schedule first 5 social clip posts (start teasing 2 weeks before launch)

---

## Phase 3: Launch + Distribution (Days 61-90)
*Go live; activate all channels*

### Launch Week
- [ ] Audible listing goes live (verify ACX approval came through)
- [ ] KDP ebook + paperback live
- [ ] IngramSpark titles live (may lag 4-6 weeks -- submit early)
- [ ] Post launch clips on TikTok + IG Reels 3x/week for first 2 weeks
- [ ] Email list blast (even if small -- build the habit)
- [ ] Ask readers for Audible reviews (reviews in first 30 days boost algorithm)
- [ ] Submit to BookBub Featured Deal (free; curated newsletter; high-conversion)
- [ ] Reach out to 3-5 book bloggers / BookTok creators for ARC copies

### Post-Launch (Days 75-90)
- [ ] Pull first sales data (KDP dashboard + ACX dashboard)
- [ ] Identify top-performing social clip (views + saves) -- make more like it
- [ ] Run KDP Countdown Deal or Free Days promo at day 30 mark (boosts rank)
- [ ] Review royalty income: if royalty share narrator, calculate break-even vs PFH
- [ ] Decide: sequence social clips for 90-day drip OR consolidate into highlight reel
- [ ] Begin planning Book 2 or companion content (sequel, workbook, extended audio)

## The Empire Extensions (Post-90-Day)

Once the first title has traction, the brand engine expands:

1. **Book 2** -- same narrator, same voice, same audience. Series build = Audible algorithm gold.
2. **Merch** -- quotes from the book on Printful/Printify. Low cost, passive.
3. **Audio Drama** -- full cast version with sound design. Higher production cost; higher perceived value.
4. **Podcast / Serial** -- release chapter-by-chapter as a free podcast to build audience for Book 2.
5. **Licensing** -- if the IP gets traction, option it for screen. Keep rights; never sell the IP outright.
