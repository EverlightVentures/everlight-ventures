## ========== SECTION 3: ABOUT ==========

### Copy
Everlight Ventures started with one idea: stop waiting for permission.

What began as a logistics company grew into a portfolio of real businesses -- each one built from scratch, funded internally, and operated with the same core principle. Move fast. Stay lean. Ship product.

The ventures span industries because the skill set does. Commerce, software, publishing, logistics -- different markets, same playbook. Find the gap. Build the solution. Scale what works. Kill what doesn't.

This is not a fund. This is not a holding company collecting logos. Every venture under the Everlight umbrella is actively operated, actively growing, and built to last.

Everlight Logistics LLC is the legal entity behind the portfolio. Headquartered in the U.S. and built for the long game.

## ========== DESIGN NOTES FOR LOVABLE ==========

### Color Palette
- Background Primary: #0A0A0A (near black)
- Background Secondary: #141414 (card surfaces)
- Text Primary: #F5F5F5 (off-white)
- Text Secondary: #A0A0A0 (muted gray for subtitles)
- Accent Primary: #D4AF37 (warm gold -- used for CTAs, hover states, highlights)
- Accent Secondary: Per-venture colors listed on each card
- Border/Divider: #2A2A2A (subtle separation)

### Typography Suggestions
- Headings: Inter or Sora (clean, geometric, modern)
- Body: Inter or DM Sans
- Accent/Taglines: Consider a lighter weight or tracking-widened variant for venture taglines

### Layout Notes
- Hero: Full viewport height, centered text, single CTA button
- Ventures Grid: 2-3 column responsive grid on desktop, single column on mobile
- Each card: Subtle hover lift effect, accent color as a thin top border or glow
- About: Single column, generous whitespace, left-aligned text
- Footer: Simple, dark, minimal -- gold accent on key links

### Interactions
- Venture cards: On hover, accent color top-border animates in, card lifts 2-4px with soft shadow
- CTA buttons: Gold fill with dark text, slight scale on hover
- Smooth scroll between sections
- Consider a subtle parallax on the hero light streak
