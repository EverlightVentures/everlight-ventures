# Everlight GPT Ask Workflow – Instruction Manual

## Setup Steps

1. Import `everlight_gpt_ask.json` into your n8n dashboard.
2. Add your Slack OAuth2 credentials under Credentials > Slack API.
3. Use the `/ask_gpt` command in any connected Slack channel.
4. Responses will be routed via selected GPT agent (Thor, Braniac, etc.).
5. You can customize the 'Switch' node to route to additional AI models.
6. All responses post back to Slack and can be optionally saved to Google Sheets.

Enjoy your new mobile GPT assistant hub!
